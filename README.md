## Install dependencies

```sh
npm i
```

Get AirMonitor Windows / Mac OS app.

## Run on dev mode

```sh
npm start
```

### The electron folder is created in the project director and applications will be built there.

### Build Windows

```sh
npm run win
```

### Build Mac

```sh
npm run mac
```
